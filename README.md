# Лабораторная работа №1

## Условие
Дана выборка объема $n=20$. По ней необходимо:

1. Приблизить математическое ожидание и дисперсию.
2. Построить вариационный ряд и эмпирическую функцию распределения (график).
3. Найти медиану.
4. Построить гистограмму выборки.
5. Сделать выводы.
